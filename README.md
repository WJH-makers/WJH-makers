## 👋 关于我

我是 **万佳泓 / WJH-makers**，武汉大学计算机科学与技术本科生（2022–2026）。

把 **AI · 系统 · 全栈 · 硬件** 串成一条可交付链路：模型训练 → API → 前端 → 部署 → 复盘文档。原则很简单——**能复现、能测试、能上线、能讲清楚**。

| 主线 | 内容 |
|:---:|:---|
| 🛰️ 毕设 | 高分辨率遥感影像 **VQA**（全栈应用 + 推理/训练链路） |
| 🤖 AI | MoE、可学习路由、多智能体；PyTorch / MindSpore / DeepSpeed |
| ⚙️ 系统 | 编译器前端 · xv6 · 五级流水线 CPU（Vivado/FPGA） |
| 🧰 工程 | Git / SSH / Docker / CI/CD · 学习日志 · Typora 主题 |

- 🌐 站点：[wwjjhh.online](https://wwjjhh.online)（学习博客自托管）
- 📝 仓库：[wjh-makers-learning-blog](https://github.com/WJH-makers/wjh-makers-learning-blog) · Vercel 镜像
- 🎨 主题：[typora-theme-claude-like](https://github.com/WJH-makers/typora-theme-claude-like)

> 🔒 私有仓库只抽象公开为经验（含 RSVQA 全栈、博客迭代、服务器运维）；不在 README 暴露密钥、内网端口或业务细节。

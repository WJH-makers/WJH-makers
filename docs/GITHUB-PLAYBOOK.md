# WJH-makers GitHub Playbook

This profile repository links the public-facing GitHub system for WJH-makers.

## Public surfaces

- Profile README: `WJH-makers/WJH-makers`
- Default community files and workflow templates: `WJH-makers/.github`
- New project README template: `WJH-makers/readme-template`

## Loaded GitHub techniques

### Discoverability

- Profile README with project map, stack summary, dynamic contribution widgets, and pinned project links.
- Repository topics and concise descriptions for search and portfolio scanning.
- README templates that force a clear problem statement, quick start, test commands, and deployment notes.

### Collaboration

- Structured issue forms for bugs, features, docs, and questions.
- Pull request template with verification evidence and secret-safety checks.
- Default `CONTRIBUTING.md`, `SUPPORT.md`, `SECURITY.md`, and `CODE_OF_CONDUCT.md` through `.github`.

### Automation

- Starter workflow templates for Node/Python, Java Maven, .NET, CodeQL, dependency review, and README health checks.
- Dependabot templates for GitHub Actions and common ecosystems.
- Profile-specific README health workflow to catch missing local assets and broken internal references.

### Security

- Security policy that routes vulnerabilities away from public issues.
- Dependabot and dependency review templates.
- Secret hygiene checklist: no `.env`, tokens, private keys, database URIs, or private URLs in public repos.
